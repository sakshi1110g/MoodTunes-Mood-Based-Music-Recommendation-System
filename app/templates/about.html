Technologies Used

For this project, I utilized a range of technologies to bring our vision to life. On the backend, I employed Flask and Python to handle server-side logic and data processing. The frontend was built using HTML, CSS, and JavaScript to ensure an intuitive and visually appealing user interface. Additionally, I integrated machine learning tools such as OpenCV, Pandas, and NumPy to enhance the project's capabilities, enabling advanced features like mood detection and personalized recommendations. This combination of technologies allowed us to create a robust and dynamic web application.
